# Job Search Agent

An AI-powered job search agent that finds best-fit jobs for you every day and delivers them to your inbox by morning.

Built with [Claude Code](https://claude.com/product/claude-code). No coding experience required.

## What It Does

- Searches public job boards overnight (LinkedIn, Indeed, Glassdoor, and more)
- Reads every listing and scores it against your qualifications, preferences, and deal-breakers
- Sends you one email each morning with only the jobs worth looking at
- Learns what you want through a detailed profile — not just keywords, but context and fit

## Who This Is For

You're deep in a job search and tired of LinkedIn filters returning hundreds of irrelevant results. You know exactly what you're looking for, but the tools don't let you get that specific.

This agent does.

## What You'll Need

- A Mac (Windows/Linux should work too with minor adjustments)
- [Claude Code](https://claude.com/product/claude-code) installed
- An [Anthropic API key](https://console.anthropic.com) (~$0.50–$2.00/day for job scoring)
- A free email service account ([Resend](https://resend.com) recommended)
- 1–2 hours to set it up

## How It Started

I built this because I was looking for something very specific — a founding marketer role at a B2B startup building from 0 to 1. LinkedIn couldn't filter for that. So I built something that could.

The first morning it ran, I woke up to 17 matches. Every one of them relevant.

I'd never used Terminal before that weekend.
